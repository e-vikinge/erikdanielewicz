Nook Crossing

Nook Crossing is a fan game based off of Nintendo's Animal Crossing series that I developed solo. It is a chapter-based game that I made with the Godot game engine. Aside from learning GDScript, (Godot's high-level, dynamically typed programming language) I created all pixel-art assets, implemented story dialog, and coded all logic. The game takes about 15 minutes and will immediately start with sound when clicking the "Find Out More" button below. Use the arrow keys to move and left mouse click to advance dialog.

Deku Web Scraper

This application retrieves Nintendo Switch game sale data from https://www.dekudeals.com/ and displays it in a tabular format. While the most current deals are displayed, previous pages can be displayed by changing the number at the end of the URL. The cells in the Link column can be copied by using the shortcut: Ctrl + C. This application is written in C# and XAML.
